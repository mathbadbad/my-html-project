# 🌙 眠音 - AI助眠助手 v3.0

> 一个基于Web Audio API和语音合成的智能助眠应用，采用潮汐App风格设计，支持人声引导冥想、治疗频率音效和智能唤醒。

## ✨ 核心功能

### 🗣️ 人声引导冥想
- **身体扫描** (10分钟) - 从头到脚的深度放松引导
- **呼吸放松** (5分钟) - 4-7-8呼吸法专业指导
- **感恩练习** (8分钟) - 睡前 gratitude journal 引导
- **睡前故事** (15分钟) - 沉浸式星空草原冥想

### 🎙️ 语音自定义
- 温柔女声 / 沉稳男声 切换
- 语速调节 (0.5x - 1.0x)
- 音量独立控制
- 实时语音波形可视化
- 呼吸引导动画同步

### 🎵 治疗音频
- **432Hz** - 自然和谐频率
- **528Hz** - 修复与转化
- **174Hz** - 深层放松
- **4-8Hz** - 脑波诱导 (Theta/Alpha)
- **双耳节拍** - 科学助眠
- **粉噪音/棕噪音** - 屏蔽干扰

### 🌅 智能唤醒
- 4种唤醒模式：温柔曙光、自然苏醒、活力唤醒、智能分析
- 浅睡期唤醒算法（模拟）
- 渐进音量增强
- 手机震动提醒

### 🎨 设计特色
- 潮汐App风格极简UI
- 雾蓝/沙白/森林绿配色
- 呼吸动画背景
- 毛玻璃效果
- 全屏沉浸播放器

## 🚀 使用方法

### 1. 直接运行
```bash
双击打开 mianyin-app-v3.html
```

### 2. 使用步骤
1. 选择场景（午睡/夜间）
2. 选择人声冥想主题（可选）
3. 调整语音设置（可选）
4. 填写今日状态（5个维度）
5. 设置唤醒时间（可选）
6. 点击"开始睡眠旅程"

### 3. 浏览器支持
- ✅ Chrome / Edge / Safari（推荐）
- ✅ 支持移动端浏览器
- ✅ 支持PWA安装（添加到主屏幕）
- ⚠️ 语音合成功能需要网络连接（首次加载）

## 📱 安装为App（PWA）

### iOS Safari
1. 用Safari打开页面
2. 点击"分享"按钮
3. 选择"添加到主屏幕"
4. 即可像原生App一样使用

### Android Chrome
1. 用Chrome打开页面
2. 点击菜单（三个点）
3. 选择"添加到主屏幕"
4. 或"安装应用"

## 🔧 技术原理

### Web Audio API
- 实时生成治疗频率
- 双耳节拍（Binaural Beats）技术
- 动态噪音生成（粉/棕噪音）
- 低通滤波器柔化处理

### Web Speech API
- 浏览器原生语音合成
- 中文语音支持
- 语速/音调实时调节
- 多声音选择

### 冥想脚本
基于正念减压（MBSR）和认知行为疗法（CBT）原理设计：
- 身体扫描法（Body Scan）
- 4-7-8呼吸法（Dr. Weil）
- 感恩干预（Gratitude Intervention）
- 引导意象（Guided Imagery）

## ⚠️ 注意事项

1. **语音合成**：首次使用可能需要联网加载语音包
2. **后台播放**：部分手机锁屏后可能暂停，建议保持屏幕常亮或允许后台音频
3. **耳机推荐**：使用立体声耳机可获得最佳双耳节拍效果
4. **音量控制**：建议音量保持在30-60%，保护听力
